Промпт для Replit (дверь, кнопка-ручка, поиск)

В разделе «Совещания» (страница /meetings) нужно полностью изменить внешний вид списка комнат.

1. Визуал — настоящая дверь

Каждая комната отображается как стилизованная дверь (прямоугольник с текстурой дерева или металла, с тенью, с «ручкой»).

· На двери — табличка (как на офисах), на которой написано:
  · Номер офиса (например, «Офис 982864»)
  · Название планёрки (например, «Мой тест»)
  · Имя создателя / руководителя (например, «директор: Иван Иванов»)
· Ручка двери — это кнопка «Войти». Сделать её похожей на дверную ручку: горизонтальная или вертикальная планка, при нажатии происходит переход в комнату.
· Дополнительные кнопки «Копировать ссылку» и «Поделиться» разместить маленькими иконками рядом с ручкой или под табличкой (чтобы не портили образ).

2. Функциональность

· Поиск над списком дверей: поле ввода с иконкой лупы, фильтрация по номеру офиса, названию, имени создателя.
· Кнопка «Войти» (ручка) ведёт в комнату по ссылке-приглашению (если есть токен) или на страницу входа.
· Создание новой комнаты — остаётся отдельной кнопкой, не в виде двери.

3. Технически

· Использовать CSS (или Tailwind) для создания эффекта двери: фон, градиенты, тени, псевдоэлемент ::before для ручки.
· При наведении — дверь может слегка приоткрываться (трансформация, тень).
· Данные для таблички брать из объекта комнаты: meetingId (номер офиса), name (название), creatorName (имя создателя — нужно подтягивать из связанной таблицы accounts).

4. Примерное расположение

```
🔍 Поиск по номеру, названию или имени руководителя

┌─────────────────────────────────┐
│                                 │
│   ┌───────────────────────┐     │
│   │ ОФИС 982864            │     │
│   │ Мой тест               │     │
│   │ директор: Иван Иванов  │     │
│   └───────────────────────┘     │
│                                 │
│           🚪 (ручка-кнопка)      │
│                                 │
└─────────────────────────────────┘
```

(Ручка — кликабельный элемент, остальная часть двери — нет.)

Пожалуйста, реализуйте такое оформление.
